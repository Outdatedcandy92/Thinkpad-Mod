# Thinkpad Mod

A small board that I designed to be embedded inside my Thinkpad X1 Carbon Gen 7 laptop. It features a MCU on board along with a ton of sensors for data logging and an RGB led placed to shine on the red dot that's on the Thinkpad Logo.
